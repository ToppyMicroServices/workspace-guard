# Support

## Getting Help

- Usage questions and bug reports: [GitHub Issues](https://github.com/ToppyMicroServices/workspace-guard/issues)
- Security issues: [Security Policy](.github/SECURITY.md)

## What To Include

- Workspace Guard version
- VS Code version
- Operating system
- Reproduction steps
- Expected behavior
- Actual behavior

Please do not include secrets, tokens, private keys, or private credentials in public issues.
