# Code of Conduct

This project aims to be respectful, practical, and safe for contributors and users.

## Our Standards

- Be respectful and assume good intent.
- Focus on facts, code, and user impact.
- Avoid harassment, intimidation, and personal attacks.
- Do not pressure others into unsafe commands, unsafe disclosures, or hostile behavior.

## Scope

This applies to issues, pull requests, code review, release discussions, and other project spaces.

## Reporting

If you experience or observe unacceptable behavior:

- Use GitHub's reporting tools for urgent abuse or platform-level misconduct.
- For project-level moderation follow-up, open a limited-detail issue asking maintainers to review the situation.

Please avoid posting private or sensitive details publicly.
